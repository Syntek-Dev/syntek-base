# project-management/src/08-GDPR/PLANNING

GDPR gap analysis reports generated by scanning stories, sprints, database schema, and user flow
documents against `project-management/docs/GDPR-GUIDE.md`.

## Directory Tree

```text
project-management/src/08-GDPR/PLANNING/
├── CONTEXT.md                    ← this file
├── GDPR-REPORT-STORIES.md        ← per-story GDPR analysis             [created by planning workflow]
├── GDPR-REPORT-SPRINTS.md        ← per-sprint GDPR analysis             [created by planning workflow]
├── GDPR-REPORT-DATABASE.md       ← encryption, retention, and RLS audit [created by planning workflow]
└── GDPR-REPORT-USER-FLOWS.md     ← per-flow GDPR gap reports            [created by planning workflow]
```

## When to read these

- Before implementing any feature that touches personal data
- Before running `workflows/08-gdpr-compliance/`
- When assessing the GDPR scope of a new story or sprint
- When preparing a data inventory, lawful basis register, or sub-processor register

## Reports

| File                        | Source scanned      | Coverage                                                                              |
| --------------------------- | ------------------- | ------------------------------------------------------------------------------------- |
| `GDPR-REPORT-STORIES.md`    | `src/01-STORIES/`   | Per-story: PII present, articles triggered, what is covered, what is missing          |
| `GDPR-REPORT-SPRINTS.md`    | `src/02-SPRINTS/`   | Per-sprint: PII stories, GDPR tasks present, systemic gaps                            |
| `GDPR-REPORT-DATABASE.md`   | `src/03-DATABASE/`  | Encryption, retention, erasure paths, RLS, consent records, cross-file contradictions |
| `GDPR-REPORT-USER-FLOWS.md` | `src/04-USER-FLOW/` | Per-flow: PII collected, GDPR steps present, transparency and lawful basis gaps       |

These reports are generated by running the GDPR planning workflow against the current state of each
source folder. They have not yet been created — run `workflows/08-gdpr-compliance/` to produce them.
