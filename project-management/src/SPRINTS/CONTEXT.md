# project-management/src/SPRINTS

Sprint plans for the project-name project.

## Directory Tree

```text
project-management/src/SPRINTS/
├── CONTEXT.md               ← this file
└── SPRINT-##.md             ← sprint plans (e.g. SPRINT-01.md, SPRINT-02.md …)
```

**Naming:** `SPRINT-##.md` — 2-digit (e.g. `SPRINT-01.md`)

Each sprint file contains: goal, stories included (MoSCoW), timeline.
