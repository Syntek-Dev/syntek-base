# User Story Creation — Steps

**Last Updated**: 18/04/2026 **Version**: 1.0.0 **Maintained By**: Syntek Studio
**Language**: British English (en_GB)

---

## Steps

### Step 1 — Generate the Story

```text
/syntek-dev-suite:stories [describe the feature and user role]
```

### Step 2 — Review and Edit

Review the generated story for:

- Clear "As a [role], I want [goal], so that [benefit]" format
- Specific, testable acceptance criteria
- No ambiguity in scope

### Step 3 — Assign Story Number

Check `project-management/src/STORIES/` for the next available number.
Name the file `US###.md`.

### Step 4 — Save

Save to `project-management/src/STORIES/US###.md`.

### Step 5 — Commit

```text
/syntek-dev-suite:git
```
