# code/src/frontend/src/graphql/generated

Auto-generated TypeScript types, hooks, and utilities produced by GraphQL Code Generator.
**Never edit files in this directory by hand** — all changes are overwritten on the next codegen run.

## Rules

- Files here are regenerated by `pnpm run codegen` (run via the frontend container).
- Do not commit partial or stale generated files — always regenerate after a backend schema change.
- Do not add custom code to this directory.
- Import generated hooks as: `import { useGetUserQuery } from "@/graphql/generated"`.

## Regenerating

```bash
docker compose exec frontend pnpm run codegen
```
