# code/src/scripts/audits/reports

Output directory for reports generated by the audit scripts in `../`.

## Directory Tree

```text
code/src/scripts/audits/reports/
├── CONTEXT.md               ← this file
├── .gitignore               ← ignores all generated reports; keeps tracked files
├── .gitkeep                 ← keeps this folder present in git when empty
├── stubs-report.*           ← generated by stubs.sh --output <FORMAT>    [gitignored]
└── cloc-report.*            ← generated by cloc.sh --output <FORMAT>     [gitignored]
```

## Tracked Files

| File         | Purpose                                                                           |
| ------------ | --------------------------------------------------------------------------------- |
| `.gitignore` | Ignores all generated reports; keeps `.gitkeep`, itself, and `CONTEXT.md` tracked |
| `.gitkeep`   | Keeps this folder present in git when no reports exist                            |
| `CONTEXT.md` | This file                                                                         |

## Generated Reports (gitignored)

| Filename pattern                               | Source script                |
| ---------------------------------------------- | ---------------------------- |
| `stubs-report.md` / `.txt` / `.json` / `.html` | `stubs.sh --output <FORMAT>` |
| `cloc-report.md` / `.txt` / `.json` / `.html`  | `cloc.sh --output <FORMAT>`  |

Generated reports are never committed. Use `--output-file PATH` on any script to write a report
outside this directory if persistence is needed (e.g. for CI artefact upload).
