# code/src/scripts/reports

Output directory for reports generated by the quality scripts in `../`.

## Directory Tree

```text
code/src/scripts/reports/
├── CONTEXT.md               ← this file
├── .gitignore               ← ignores all generated reports; keeps tracked files
├── .gitkeep                 ← keeps this folder present in git when empty
├── lint-report.*            ← generated by lint.sh --output <FORMAT>       [gitignored]
├── check-report.*           ← generated by check.sh --output <FORMAT>      [gitignored]
└── format-report.*          ← generated by format.sh --output <FORMAT>     [gitignored]
```

## Tracked Files

| File         | Purpose                                                                           |
| ------------ | --------------------------------------------------------------------------------- |
| `.gitignore` | Ignores all generated reports; keeps `.gitkeep`, itself, and `CONTEXT.md` tracked |
| `.gitkeep`   | Keeps this folder present in git when no reports exist                            |
| `CONTEXT.md` | This file                                                                         |

## Generated Reports (gitignored)

| Filename pattern                                | Source script                 |
| ----------------------------------------------- | ----------------------------- |
| `lint-report.md` / `.txt` / `.json` / `.html`   | `lint.sh --output <FORMAT>`   |
| `check-report.md` / `.txt` / `.json` / `.html`  | `check.sh --output <FORMAT>`  |
| `format-report.md` / `.txt` / `.json` / `.html` | `format.sh --output <FORMAT>` |

Generated reports are never committed. Use `--output-file PATH` on any script to write a report outside this directory if persistence is needed (e.g. for CI artefact upload).
