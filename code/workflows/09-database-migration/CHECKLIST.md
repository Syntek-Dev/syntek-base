# Django Database Migration — Checklist

**Last Updated**: 18/04/2026 **Version**: 1.0.0 **Maintained By**: Syntek Studio
**Language**: British English (en_GB)

---

## Execution Checklist

- [ ] Migration generated and reviewed — matches intended schema change
- [ ] Migration applied successfully to dev database
- [ ] `showmigrations` confirms clean state
- [ ] All existing tests still pass after migration
- [ ] No data loss for existing records (if altering existing fields)

---

## Definition of Done

- [ ] Migration file committed alongside model changes
- [ ] Tests green
- [ ] Committed and pushed
