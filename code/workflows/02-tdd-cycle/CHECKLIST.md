# TDD Cycle — Checklist

**Last Updated**: 18/04/2026 **Version**: 1.0.0 **Maintained By**: Syntek Studio
**Language**: British English (en_GB)

---

## Pre-Conditions

- [ ] Acceptance criteria are clear and agreed
- [ ] Test runner is available and working

---

## Execution Checklist

- [ ] Tests written before any implementation
- [ ] All new tests were red before proceeding to Green
- [ ] Minimum implementation written — no extras
- [ ] All tests are green
- [ ] Refactor completed — no logic changes
- [ ] Tests still green after refactor
- [ ] Coverage floor met (75% backend, 70% frontend)

---

## Definition of Done

- [ ] All tests pass with real implementation (no stubs)
- [ ] Coverage floor not dropped
- [ ] Committed and pushed
