# Debugging — Checklist

**Last Updated**: 18/04/2026 **Version**: 1.0.0 **Maintained By**: Syntek Studio

## Execution Checklist

- [ ] Root cause identified and documented
- [ ] Fix implemented and tested
- [ ] Tests passing (no regressions introduced)
- [ ] If a significant bug: bug report saved to `project-management/src/BUGS/`

---

## Definition of Done

- [ ] The problem no longer occurs
- [ ] Tests confirm the fix
- [ ] Committed and pushed
